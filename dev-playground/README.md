# dev-playground

This folder contains the Vite + React build tooling used during development.

**These files are NOT part of the Chrome/Edge extension.**

The loadable extension is in the [`extension/`](../extension/) folder — load that
directory directly in Chrome or Edge (developer mode, "Load unpacked").

## Dev build

```bash
cd dev-playground
npm install
npm run build   # output goes to ../dist/
```
